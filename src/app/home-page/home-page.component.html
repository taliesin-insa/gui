Taliesin est une application web de génération de bases d'apprentissage pour les reconnaisseurs d'écriture manuscrite.

Grâce à Taliesin, vous gagnerez un temps considérable sur votre travail d'annotateur. L'interface a été prévue pour être la plus ergonomique possible afin de ne plus perdre une seule seconde. De plus, l'application intègre un reconnaisseur d'écriture manuscrite pour vous proposer des transcriptions de texte que vous n'aurez plus qu'à valider ou corriger.

Si vous souhaitez en savoir plus sur ce projet, vous pouvez lire la présentation sur ce site.
